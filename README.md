# Cyber Nexus

Cyber Nexus is a futuristic, responsive landing page for a fictional human-augmentation and cybernetic technology brand. The project presents a polished product experience with a 3D animated hero, interactive motion effects, technology showcases, product cards, an AI assistant demo, animated statistics, testimonials, and a contact flow.

This project was built as a modern web interface concept for an IIT Tech Fest submission, focusing on immersive UI, accessibility, responsive design, and frontend engineering quality.

## Project Description

Cyber Nexus imagines the future of human-machine integration through a sleek product landing page. It highlights advanced neural interfaces, AI co-processors, cybernetic vision systems, and a conversational AI assistant named NEXUS.

The goal of the project is to demonstrate how emerging technology concepts can be communicated through an engaging, high-performance web experience. The interface combines 3D graphics, micro-interactions, structured content sections, and mobile-friendly navigation to create a complete landing page suitable for a futuristic tech product.

## Key Features

- Interactive 3D hero section built with React Three Fiber and Drei
- Futuristic Cyber Nexus branding and responsive navigation
- Technology showcase for neural interfaces, cyber vision, AI cortex, and quantum processing
- Product section featuring fictional cybernetic products
- Animated AI assistant demo conversation
- Motion-based scroll reveal animations using Framer Motion
- Animated impact statistics and testimonials
- Contact form with basic validation and success state
- Accessible skip link, semantic sections, ARIA labels, and reduced-motion handling
- Fully responsive design for desktop, tablet, and mobile screens

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open the app in your browser:

```text
http://localhost:3000
```

Build for production:

```bash
pnpm build
```

Start the production server:

```bash
pnpm start
```
